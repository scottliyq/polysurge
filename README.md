## 🧠 核心算法

### 智能异常检测
每30秒分析100个高流动性市场，识别5种关键异常：

| 异常类型 | 检测标准 | 信号强度 |
|---------|---------|---------|
| 🔥 **成交量飙升** | 5分钟成交量 > 平均值5倍 | 高 |
| 🐋 **大额交易** | 单笔交易 > 平均值20倍 | 极高 |
| 📈 **价格异动** | 5分钟波动 > 10% | 高 |
| ⚖️ **买卖失衡** | 单边交易 > 85% | 中等 |
| ⚡ **高频交易** | 5分钟内 > 10笔交易 | 中等 |

### 智能评分系统 (0-100分)

**不是简单的计数，而是多维度智能评估：**

```
评分 = 基础权重 × 叠加系数 × 市场流动性 × 异常强度

基础权重：大额(40) > 成交量(30) > 价格(20) > 失衡(15) > 高频(10)
叠加系数：多种异常同时出现获得额外加分
市场流动性：高流动性市场信号更有价值
异常强度：成交量倍数、价格波动幅度等
```

**评分等级：**
- 🔴 **70分以上** = 强信号（立即关注）
- 🟡 **40-69分** = 中等信号（值得关注）  
- ⚪ **40分以下** = 弱信号（仅供参考）

### 信号验证系统

**实时追踪每个信号后的价格变化：**
- ✅ **绿色↑** - 信号后5分钟价格上涨
- ❌ **红色↓** - 信号后5分钟价格下跌

帮你判断哪种信号更可靠，持续优化算法。

## 🎮 产品功能

### 📊 实时数据面板
- **监控市场数** - 智能选择100个高成交量+高流动性市场
- **异常事件** - 检测到的信号总数，支持时间筛选
- **活跃钱包** - 参与异常交易的钱包地址数
- **总成交量** - 所有异常事件涉及的成交金额

### 🚨 实时异常事件流
- **智能评分** - 红色70+强信号，黄色40-69中等，灰色较弱
- **一键跳转** - 点击市场名称直达 Polymarket
- **信号验证** - 实时显示信号后5分钟价格变化
- **多维度信息** - 异常类型、成交量、价格变化一目了然

### 📈 市场情绪分析
- **整体买卖比例** - 快速判断市场多空情绪
- **强烈看涨** >65%买入 | **偏向看涨** 55-65%
- **中性** 45-55% | **偏向看跌** 35-45% | **强烈看跌** <35%

### 🏆 排行榜单
- **热门异动市场** - 异常事件最多的市场排名
- **大额交易榜** - 鲸鱼活动最活跃的市场
- **成交量飙升榜** - 放量倍数最高的市场
- **价格异动榜** - 波动最剧烈的市场
